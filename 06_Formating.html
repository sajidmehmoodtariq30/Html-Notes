Formatting Tags

Bold Tag
This tag increases the boldbess of a text or set of texts
Italic Tag
This tag emphasizes the text.
Underline tag
This tag puts an underline below the text
Superscript
This tag writes content in the form of power of something like a2
Subscript
This tag writes content in the form of base of something like H2O
Delete Tag
This tag puts a line through text
