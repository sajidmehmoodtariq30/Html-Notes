Image Tag in HTML

Image tag is used to insert an image in our website.img is the tag used while scr,allign,width and alt are its attributes

Absolute URL

Absolute refers to a full web address i.e. including https or www part

Relative URL

Relative refers to link without http or www part i.e. from local machine.

src
scr attribute is used to give path of the image it can be relative or it can be Absolute
alt
alt attribute is used to show text which appears if picture is not present.

width
width attribute is used to give height to an image by default it reads data entered in pixels. Same for length. Width can be entered in % but height should'nt be beacuse height of a website can vary
allign
Align attribute is used to align the image vertically or horizontally
